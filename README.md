使用方法：
1.编译（不需要JSON库）       
1.运行（注意不要再IDE的控制台，一定要自己使用终端来运行）      
1.输入标题     
1.输入长度     
1.是否输出到文件(y/n)     
1.enjoy!     

注意：若输出乱码，请使用在线IDE，或检查编码格式是否为UTF-8
